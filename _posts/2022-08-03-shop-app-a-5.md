# Mengubah Pubspec.yaml

----

Di sini, kita akan menggunakan sebuah package yang bernama [badges](https://pub.dev/packages/badges/example){:target="\_blank"}. Silakan tambahkan package ini pada pubspec.yaml kalian, seperti ini:

```pubspec.yaml
dependencies:
  flutter:
    sdk: flutter

  badges: ^2.0.3
  provider: ^6.0.3
  cupertino_icons: ^1.0.2
```

# Membuat Cart Model

----

Selanjutnya, kita akan membuat cart model. Pada folder `models`, buat sebuah file baru dengan nama `cart_model.dart`, sesuaikan source code-nya menjadi seperti ini:

```dart
class CartModel {
  final String id;
  final String name;
  final double price;
  final int quantity;

  CartModel({
    required this.id,
    required this.name,
    required this.price,
    required this.quantity,
  });
}
```

# Membuat Cart Provider

----

Selanjutnya, pada folder `providers`, buat sebuah file baru dengan nama `cart_provider.dart`, sesuaikan source code-nya menjadi seperti ini:

```dart
import 'package:flutter/material.dart';

import '../models/cart_model.dart';
import '../models/product_model.dart';

class CartProvider extends ChangeNotifier {
  Map<String, CartModel> _carts = {};

  Map<String, CartModel> getCarts() {
    return _carts;
  }

  int getCartLength() {
    return _carts.length;
  }

  void addProductToCart(ProductModel product) {
    if (_carts.containsKey(product.id)) {
      _carts.update(
        product.id,
        (value) => CartModel(
          id: value.id,
          name: value.name,
          price: value.price,
          quantity: value.quantity + 1,
        ),
      );
    } else {
      _carts.putIfAbsent(
        product.id,
        () => CartModel(
          id: DateTime.now().toString(),
          name: product.name,
          price: product.price,
          quantity: 1,
        ),
      );
    }

    notifyListeners();
  }
}
```

Di sini, kita akan menyimpan data `carts` dalam bentuk map. Key yang akan digunakan berasal dari produk id dan value-nya adalah cart model itu sendiri.

Ketika menambahkan produk pada cart, maka kita lakukan pengecekan. Apakah produk tersebut sudah di dalam cart atau belum. Jika belum ada, maka tambahkan produk, jika sudah ada, maka quantity-nya bertambah 1. 

Selain itu, kita juga memiliki sebuah fungsi bernama `getCartLength` untuk mendapatkan panjang dari `carts`.

# Mengubah Product Widget

----

Selanjutnya, buka `product_widget.dart`, lalu sesuaikan source code-nya menjadi seperti ini:

```dart
import 'package:flutter/material.dart';
import 'package:provider/provider.dart';

import '../models/product_model.dart';
import '../providers/cart_provider.dart';
import '../providers/product_provider.dart';
import '../screens/product_detail_screen.dart';

class ProductWidget extends StatelessWidget {
  final ProductModel product;

  const ProductWidget({
    Key? key,
    required this.product,
  }) : super(key: key);

  @override
  Widget build(BuildContext context) {
    final cartProvider = Provider.of<CartProvider>(context, listen: false);

    return ClipRRect(
      borderRadius: BorderRadius.circular(10.0),
      child: GridTile(
        footer: GridTileBar(
          backgroundColor: Colors.black54,
          leading: Consumer<ProductProvider>(builder: (context, value, child) {
            return IconButton(
              onPressed: () {
                context.read<ProductProvider>().toggleFavorite(product.id);
              },
              icon: Icon(
                product.isFavorite ? Icons.favorite : Icons.favorite_border,
                color: Colors.blue.shade300,
              ),
            );
          }),
          title: Text(product.name),
          trailing: IconButton(
            onPressed: () {
              cartProvider.addProductToCart(product);
            },
            icon: Icon(
              Icons.shopping_cart,
              color: Colors.blue.shade300,
            ),
          ),
        ),
        child: GestureDetector(
          child: Image.network(product.imageUrl),
          onTap: () {
            Navigator.pushNamed(
              context,
              ProductDetailScreen.routeName,
              arguments: product.id,
            );
          },
        ),
      ),
    );
  }
}
```

Perhatikan, kita memiliki `cartProvider`. Di sini, kita buat `listen: false` karena tidak ada nilai yang perlu diperhatikan / di-rebuild.

Kemudian, pada icon cart, kita panggil fungsi yang sudah kita buat sebelumnya, yaitu `addProductToCart`.

# Mengubah Product Screen

----

Di sini, kita akan menggunakan `badges` yang sudah kita install sebelumnya. Silakan sesuaikan source code-nya menjadi seperti ini:

```dart
import 'package:badges/badges.dart';
import 'package:flutter/material.dart';
import 'package:provider/provider.dart';

import '../providers/cart_provider.dart';
import '../widgets/product_grid_widget.dart';

enum SelectedProductFilter {
  all,
  favoriteOnly,
}

class ProductScreen extends StatefulWidget {
  const ProductScreen({Key? key}) : super(key: key);

  @override
  State<ProductScreen> createState() => _ProductScreenState();
}

class _ProductScreenState extends State<ProductScreen> {
  bool selectedProductFilter = false;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        centerTitle: true,
        title: const Text("Shop App"),
        actions: [
          PopupMenuButton(
            onSelected: (SelectedProductFilter filter) {
              setState(() {
                if (filter == SelectedProductFilter.all) {
                  selectedProductFilter = false;
                } else {
                  selectedProductFilter = true;
                }
              });
            },
            icon: const Icon(Icons.more_horiz),
            itemBuilder: ((context) {
              return [
                const PopupMenuItem(
                  value: SelectedProductFilter.all,
                  child: Text("All"),
                ),
                const PopupMenuItem(
                  value: SelectedProductFilter.favoriteOnly,
                  child: Text("Favorite Only"),
                ),
              ];
            }),
          ),
          Consumer<CartProvider>(builder: ((context, value, child) {
            final cartLength =
                context.read<CartProvider>().getCartLength().toString();

            return Badge(
              badgeContent: Text(
                cartLength,
                style: const TextStyle(
                  color: Colors.black87,
                ),
              ),
              position: BadgePosition.topEnd(top: 5, end: 15),
              badgeColor: Colors.amber.shade200,
              child: const Icon(Icons.shopping_cart),
            );
          }))
        ],
      ),
      body: ProductGridWidget(
        selectedProductFilter: selectedProductFilter,
      ),
    );
  }
}
```

Perhatikan, di sini kita menggunakan `Consumer`. Tujuannya adalah agar icon bisa di-rebuild setiap kali produk ditambahkan ke cart.

Kemudian, kita panggil fungsi yang sudah kita definisikan sebelumnya, yaitu `getCartLength` dan menampilkannya pada `badges`.

# Menambahkan Provider 

---

Buka `main.dart`, lalu sesuaikan source code-nya menjadi seperti ini:

```dart
import 'package:flutter/material.dart';
import 'package:provider/provider.dart';

import '../screens/product_screen.dart';
import '../providers/cart_provider.dart';
import '../providers/product_provider.dart';
import '../screens/product_detail_screen.dart';

void main() {
  runApp(
    MultiProvider(
      providers: [
        ChangeNotifierProvider(
          create: ((context) => ProductProvider()),
        ),
        ChangeNotifierProvider(
          create: ((context) => CartProvider()),
        )
      ],
      child: const MyApp(),
    ),
  );
}

class MyApp extends StatelessWidget {
  const MyApp({Key? key}) : super(key: key);

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      debugShowCheckedModeBanner: false,
      title: "Shop App",
      home: const ProductScreen(),
      routes: {
        ProductDetailScreen.routeName: (context) => const ProductDetailScreen()
      },
    );
  }
}
```

Perhatikan pada `MultiProvider`, kita memiliki dua provider, yaitu `ProductProvider` dan `CartProvider`.
