# Pendahuluan

---

Halo! Pada tutorial ini, kita akan membuat sebuah aplikasi yang bernama Shop App. Chapter A akan berfokus pada state management. Berikut adalah fitur yang akan dibuat pada chapter ini:

1. Menampilkan seluruh item/produk
2. Menambahkan item/produk pada cart
3. Menghapus item/produk pada cart
4. Halaman yang berbeda (Shop, Orders)
5. Menambahkan item/produk yang ada di cart pada halaman Orders

# Membuat Project

---

Baik, tanpa berlama-lama, silakan buat sebuah projek baru dengan perintah berikut ini:

```bash
flutter create shop_app
```

Jika sudah, silakan buka projek tersebut. Kemudian, buat folder dengan nama models, screens dan widgets pada folder lib, seperti gambar di bawah ini.

![Projek Awal](https://i.postimg.cc/Kvm5dgj4/1.png)

# Membuat Product Model

---

Klik pada folder models, lalu buat sebuah file baru dengan nama product_model.dart. Ini adalah model dari produk yang akan kita buat nantinya. Sesuaikan source code-nya menjadi seperti ini:

```dart
class ProductModel {
  final String id;
  final String name;
  final String description;
  final double price;
  final String imageUrl;

  bool isFavorite;

  ProductModel({
    required this.id,
    required this.name,
    required this.description,
    required this.price,
    required this.imageUrl,
    this.isFavorite = false,
  });
}
```

Setiap produk memiliki id, nama, deskripsi, harga, url gambar dan variable boolean is favorite. Kenapa menggunakan url untuk gambar? Kenapa tidak disimpan sebagai asset? Tujuannya adalah agar aplikasi tidak perlu dipublikasikan ulang setiap kali konten ditambahkan. Variable is favorite di situ berguna sebagai penanda bahwa produk tersebut disukai oleh pengguna.

Terdapat sebuah variable dengan nama crossAxisCount. Apa gunanya? Gunanya adalah untuk menampung berapa banyak produk yang harus ditampilkan dalam satu baris.

Nah, dari sini, kita akan mengukur berapa banyak produk yang harus ditampilkan berdasarkan lebar dari layar aplikasi kita. Itu semua dilakukan dalam percabangan if yang ada pada source code di atas.

Kemudian, ada juga sebuah list dengan nama products yang berisi empat buah produk yang berbeda. Nah, crossAxisCount dan products ini akan kita pass/kirim ke sebuah widget yang bernama ProductWidget.

Di sini, kita belum membuatnya, jadi silakan ikuti langkah selanjutnya.

# Membuat Product Widget

---

Klik pada folder widgets, lalu buat sebuah file baru dengan nama product_widget.dart. Sesuaikan source code-nya menjadi seperti ini:

```dart
import 'package:flutter/material.dart';

import '../models/product_model.dart';

class ProductWidget extends StatelessWidget {
  final int crossAxisCount;
  final List<ProductModel> products;

  const ProductWidget({
    Key? key,
    required this.crossAxisCount,
    required this.products,
  }) : super(key: key);

  @override
  Widget build(BuildContext context) {
    return Padding(
      padding: const EdgeInsets.all(20.0),
      child: GridView.builder(
          gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(
            crossAxisCount: crossAxisCount,
            mainAxisSpacing: 20.0,
            crossAxisSpacing: 20.0,
            childAspectRatio: 3 / 2,
          ),
          itemCount: products.length,
          itemBuilder: (BuildContext context, int index) {
            ProductModel product = products[index];

            return ClipRRect(
              borderRadius: BorderRadius.circular(10.0),
              child: GridTile(
                footer: GridTileBar(
                  backgroundColor: Colors.black38,
                  leading: IconButton(
                    onPressed: () {},
                    icon: const Icon(
                      Icons.favorite,
                      color: Colors.blue,
                    ),
                  ),
                  title: Text(product.name),
                  trailing: IconButton(
                    onPressed: () {},
                    icon: const Icon(
                      Icons.shopping_cart,
                      color: Colors.blue,
                    ),
                  ),
                ),
                child: Image.network(product.imageUrl),
              ),
            );
          }),
    );
  }
}
```

Sebelumnya, kita sudah mengirimkan dua buah arguments pada ProductWidget, yaitu crossAxisCount dan products. Nah, agar bisa menerima arguments tersebut, maka kita perlu mendefinisikan constructor-nya, seperti yang kita lakukan di atas.

Di sini, kita menggunakan widget yang disediakan langsung oleh Flutter, yaitu GridView. MainAxisSpacing dan crossAxisSpacing adalah jarak yang kita berikan untuk setiap produk. ItemCount adalah banyaknya produk yang akan ditampilkan, kita mendapatkannya melalui panjang dari products.

Untuk mendapatkan setiap produk yang akan ditampilkan, kita bisa menggunakan index. Selanjutnya, kita bisa menggunakan GridTile untuk menampilkan produk tersebut. ClipRRect dipakai agar terdapat lengkungan pada setiap sudut produk yang kita buat.

Jangan lupa untuk meng-import setiap widget yang diperlukan. Perhatikan source code di atas. Jika program dijalankan, maka akan tampil seperti di bawah ini.
