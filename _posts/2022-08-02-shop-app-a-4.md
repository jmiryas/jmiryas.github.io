# Mengubah Product Provider 

---

Pertama, mari kita ubah `product_provider.dart` terlebih dahulu. Di sini kita akan menambahkan sebuah fungsi yang bertugas menampilkan produk-produk favorit saja. Sesuaikan source code-nya menjadi seperti ini:

      id: "id-4",

          "https://images.pexels.com/photos/585752/pexels-photo-585752.jpeg?auto=compress&cs=tinysrgb&w=1260&h=750&dpr=1",
    ),
  ];

  List<ProductModel> getProducts() {
    return _products;
  }

  List<ProductModel> getFavoriteOnlyProducts() {
    return _products.where((item) => item.isFavorite).toList();
  }

  void addProduct(ProductModel product) {
    _products.add(product);
    notifyListeners();
  }

  ProductModel findById(String id) {
    return _products.firstWhere((product) => product.id == id);
  }

  void toggleFavorite(String id) {
    ProductModel selectedProduct = findById(id);

    selectedProduct.isFavorite = !selectedProduct.isFavorite;

    notifyListeners();
  }
}

Perhatikan, pada `product_provider.dart`, kita menambahkan sebuah fungsi baru dengan nama `getFavoriteOnlyProducts`. Fungsi ini bertugas untuk mengembalikan produk-produk favorit saja.

# Mengubah Product Screen

---

Selanjutnya, kita akan mengubah `product_screen.dart`. Silakan sesuaikan source code-nya menjadi seperti ini:

```dart
import 'package:flutter/material.dart';

import '../widgets/product_grid_widget.dart';

enum SelectedProductFilter {
  all,
  favoriteOnly,
}

class ProductScreen extends StatefulWidget {
  const ProductScreen({Key? key}) : super(key: key);

  @override
  State<ProductScreen> createState() => _ProductScreenState();
}

class _ProductScreenState extends State<ProductScreen> {
  bool selectedProductFilter = false;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        centerTitle: true,
        title: const Text("Shop App"),
        actions: [
          PopupMenuButton(
            onSelected: (SelectedProductFilter filter) {
              setState(() {
                if (filter == SelectedProductFilter.all) {
                  selectedProductFilter = false;
                } else {
                  selectedProductFilter = true;
                }
              });
            },
            icon: const Icon(Icons.more_horiz),
            itemBuilder: ((context) {
              return [
                const PopupMenuItem(
                  value: SelectedProductFilter.all,
                  child: Text("All"),
                ),
                const PopupMenuItem(
                  value: SelectedProductFilter.favoriteOnly,
                  child: Text("Favorite Only"),
                ),
              ];
            }),
          )
        ],
      ),
      body: ProductGridWidget(
        selectedProductFilter: selectedProductFilter,
      ),
    );
  }
}
```

Perhatikan, di sini kita memiliki enum dengan nama `SelectedProductFilter`. Enum ini akan digunakan untuk filtering.

Kemudian, di sini kita juga mengubah class-nya dari `stateless` menjadi `statefull`. Tujuannya adalah agar bisa menggunakan `setState`.

Kita juga menambahkan `PopupMenuButton`. Tujuannya adalah agar muncul popup di bagian `AppBar`.

`PopupMenuButton` sudah menyediakan `onSelected`, sehingga apa yang dipilih oleh user sudah bisa didapatkan. Maka, kita hanya perlu melakukan pengecekan menggunakan if statement, seperti yang kita lakukan di atas.

Perhatikan juga, pada `ProductGridWidget`, kita menambahkan sebuah argument, yaitu `selectedProductFilter` yang kita dapatkan pada saat memilih popup.

Tentu argument ini belum ada, jadi mari kita tambahkan.

# Mengubah Product Grid Widget

---

Di sini, kita akan menambahkan sebuah parameter dan menambahkan beberapa pengecekan. Silakan sesuaikan source code-nya menjadi seperti ini:

```dart
import 'package:flutter/material.dart';
import 'package:provider/provider.dart';

import '../models/product_model.dart';
import '../widgets/product_widget.dart';
import '../providers/product_provider.dart';

class ProductGridWidget extends StatelessWidget {
  final bool selectedProductFilter;

  const ProductGridWidget({
    Key? key,
    required this.selectedProductFilter,
  }) : super(key: key);

  @override
  Widget build(BuildContext context) {
    int crossAxisCount = 2;
    double width = MediaQuery.of(context).size.width;

    if (width < 300) {
      crossAxisCount = 1;
    } else if (width < 576) {
      crossAxisCount = 2;
    } else if (width >= 576 && width < 768) {
      crossAxisCount = 3;
    } else if (width >= 768 && width < 992) {
      crossAxisCount = 4;
    } else {
      crossAxisCount = 6;
    }

    final productProvider = Provider.of<ProductProvider>(context);

    final products = selectedProductFilter
        ? productProvider.getFavoriteOnlyProducts()
        : productProvider.getProducts();

    return Padding(
      padding: const EdgeInsets.all(20.0),
      child: products.isEmpty
          ? const Center(
              child: Text("Tidak ada produk"),
            )
          : GridView.builder(
              gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(
                crossAxisCount: crossAxisCount,
                mainAxisSpacing: 20.0,
                crossAxisSpacing: 20.0,
                childAspectRatio: 3 / 2,
              ),
              itemCount: products.length,
              itemBuilder: (BuildContext context, int index) {
                ProductModel product = products[index];

                return ProductWidget(product: product);
              }),
    );
  }
}
```

Perhatikan, kita menambahkan sebuah parameter bernama `selectedProductFilter`.

Kemudian, kita dapatkan `products` yang harus ditampilkan dengan menggunakan ternary operator. 

Terakhir, kita lakukan pengecekan `products.isEmpty`. Jika produk masih kosong, maka tampilkan pesan "Tidak ada produk", selain itu tampilkan produk.
