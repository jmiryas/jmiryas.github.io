# Install Package

---

Ada dua package yang akan kita install, yaitu [flutter_launcher_icons](https://pub.dev/packages/flutter_launcher_icons) dan [cached_network_image](https://pub.dev/packages/cached_network_image). Pada `pubspec.yaml`, sesuaikan dependencies menjadi seperti ini:

```yaml
name: shop_app
description: Shop app.

publish_to: "none"

version: 1.0.0+1

environment:
  sdk: ">=2.17.3 <3.0.0"

dependencies:
  flutter:
    sdk: flutter

  intl: ^0.17.0
  badges: ^2.0.3
  provider: ^6.0.3
  cupertino_icons: ^1.0.2
  cached_network_image: ^3.2.1

dev_dependencies:
  flutter_test:
    sdk: flutter

  flutter_lints: ^2.0.0
  flutter_launcher_icons: ^0.10.0

flutter_icons:
  android: true
  image_path_android: "assets/icons/shop.png"
  min_sdk_android: 21
  windows:
    generate: true
    image_path: "assets/icons/shop.png"
    icon_size: 48

flutter:
  uses-material-design: true
```

Di sini, kita membuat sebuah folder baru dengan nama `assets`, di dalamnya, terdapat sebuah folder dengan nama `icons`. Folder ini dibuat untuk menampung icon yang akan kita gunakan sebagai launcher. Saya mendapatkan icon dari sebuah website yang bernama [flaticon](https://www.flaticon.com/premium-icon/shop_869441?term=shop&page=1&position=34&page=1&position=34&related_id=869441&origin=search).

![Launcher Icon](https://i.postimg.cc/L6Trgdjn/3.png)

Jika dependencies sudah ditambahkan, maka jalankan perintah berikut ini di terminal atau command line:

```bash
flutter pub get
flutter pub run flutter_launcher_icons:main
```

# Mengubah Android Manifest

---

Sekarang, buka folder `android/app/src/main`, lalu ubah file `AndroidManifest.xml`. Di sini, kita menambahkan beberapa line agar internet bisa digunakan pada aplikasi yang kita buat.

```xml
<manifest xmlns:android="http://schemas.android.com/apk/res/android"
    package="com.example.shop_app">

    <uses-permission android:name="android.permission.INTERNET" />
    <uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" />

   <application
        android:label="Shop App"
        android:name="${applicationName}"
        android:icon="@mipmap/ic_launcher">
        <activity
            android:name=".MainActivity"
            android:exported="true"
            android:launchMode="singleTop"
            android:theme="@style/LaunchTheme"
            android:configChanges="orientation|keyboardHidden|keyboard|screenSize|smallestScreenSize|locale|layoutDirection|fontScale|screenLayout|density|uiMode"
            android:hardwareAccelerated="true"
            android:windowSoftInputMode="adjustResize">
            <!-- Specifies an Android theme to apply to this Activity as soon as
                 the Android process has started. This theme is visible to the user
                 while the Flutter UI initializes. After that, this theme continues
                 to determine the Window background behind the Flutter UI. -->
            <meta-data
              android:name="io.flutter.embedding.android.NormalTheme"
              android:resource="@style/NormalTheme"
              />
            <intent-filter>
                <action android:name="android.intent.action.MAIN"/>
                <category android:name="android.intent.category.LAUNCHER"/>
            </intent-filter>
        </activity>
        <!-- Don't delete the meta-data below.
             This is used by the Flutter tool to generate GeneratedPluginRegistrant.java -->
        <meta-data
            android:name="flutterEmbedding"
            android:value="2" />
    </application>
</manifest>
```

Perhatikan pada `uses-permission`, kita tambahkan dua line tersebut agar aplikasi bisa menggunakan internet.

# Mengubah Product Widget

---

Di sini, kita akan menggunakan cached network image agar gambar yang berasal dari internet bisa disimpan di dalam cached. Ini akan membantu performa aplikasi kita.

Sesuaikan source code-nya menjadi seperti ini:

```dart
import 'package:flutter/material.dart';
import 'package:provider/provider.dart';
import 'package:cached_network_image/cached_network_image.dart';

import '../models/product_model.dart';
import '../providers/cart_provider.dart';
import '../providers/product_provider.dart';
import '../screens/product_detail_screen.dart';

class ProductWidget extends StatelessWidget {
  final ProductModel product;

  const ProductWidget({
    Key? key,
    required this.product,
  }) : super(key: key);

  @override
  Widget build(BuildContext context) {
    final cartProvider = Provider.of<CartProvider>(context, listen: false);

    return ClipRRect(
      borderRadius: BorderRadius.circular(10.0),
      child: GridTile(
        footer: GridTileBar(
          backgroundColor: Colors.black54,
          leading: Consumer<ProductProvider>(builder: (context, value, child) {
            return IconButton(
              onPressed: () {
                context.read<ProductProvider>().toggleFavorite(product.id);

                if (value.getProductStatus(product.id)) {
                  ScaffoldMessenger.of(context).showSnackBar(
                    const SnackBar(
                      content: Text("Produk ditambahkan ke favorit"),
                    ),
                  );
                } else {
                  ScaffoldMessenger.of(context).showSnackBar(
                    const SnackBar(
                      content: Text("Produk dihapus dari favorit"),
                    ),
                  );
                }
              },
              icon: Icon(
                product.isFavorite ? Icons.favorite : Icons.favorite_border,
                color: Colors.blue.shade300,
              ),
            );
          }),
          title: Text(product.name),
          trailing: IconButton(
            onPressed: () {
              cartProvider.addProductToCart(product);

              ScaffoldMessenger.of(context).showSnackBar(
                const SnackBar(
                  content: Text("Produk ditambahkan ke cart"),
                ),
              );
            },
            icon: Icon(
              Icons.shopping_cart,
              color: Colors.blue.shade300,
            ),
          ),
        ),
        child: GestureDetector(
          child: CachedNetworkImage(
            imageUrl: product.imageUrl,
            placeholder: (context, url) => const SizedBox(
              width: 30.0,
              height: 30.0,
              child: Center(
                child: CircularProgressIndicator(),
              ),
            ),
            fit: BoxFit.cover,
          ),
          onTap: () {
            Navigator.pushNamed(
              context,
              ProductDetailScreen.routeName,
              arguments: product.id,
            );
          },
        ),
      ),
    );
  }
}
```

Perhatikan pada `CachedNetworkImage`, di sana kita menggunakan `CircularProgressIndicator` agar ketika proses pengambilan gambar sedang berjalan, maka sebuah loading akan ditampilkan.

# Mengubah Product Detail Screen 

---

```dart
import 'package:intl/intl.dart';
import 'package:flutter/material.dart';
import 'package:provider/provider.dart';
import 'package:cached_network_image/cached_network_image.dart';

import '../providers/product_provider.dart';

class ProductDetailScreen extends StatelessWidget {
  static String routeName = "/product-detail";

  const ProductDetailScreen({Key? key}) : super(key: key);

  @override
  Widget build(BuildContext context) {
    final productId = ModalRoute.of(context)!.settings.arguments as String;

    final currentProduct =
        Provider.of<ProductProvider>(context).findById(productId);

    final numberFormat = NumberFormat("#,##0.00", "id_ID");

    double height = MediaQuery.of(context).size.height - 40.0 - 55.0;

    return Scaffold(
      appBar: AppBar(
        centerTitle: true,
        title: Text(currentProduct.name),
      ),
      body: Padding(
        padding: const EdgeInsets.all(20.0),
        child: Column(
          children: [
            SizedBox(
              width: double.infinity,
              height: 0.4 * height,
              child: ClipRRect(
                borderRadius: BorderRadius.circular(10.0),
                child: CachedNetworkImage(
                  imageUrl: currentProduct.imageUrl,
                  placeholder: (context, url) => const SizedBox(
                    width: 30.0,
                    height: 30.0,
                    child: Center(
                      child: CircularProgressIndicator(),
                    ),
                  ),
                  fit: BoxFit.cover,
                ),
              ),
            ),
            const SizedBox(
              height: 10.0,
            ),
            SizedBox(
              width: double.infinity,
              child: Text(
                currentProduct.name,
                style: const TextStyle(
                  fontWeight: FontWeight.bold,
                  fontSize: 20.0,
                ),
                overflow: TextOverflow.ellipsis,
                maxLines: 2,
              ),
            ),
            SizedBox(
              width: double.infinity,
              child: Text(
                "Rp ${numberFormat.format(currentProduct.price)}",
                style: const TextStyle(
                  fontSize: 15.0,
                ),
              ),
            )
          ],
        ),
      ),
    );
  }
}
```

Sama seperti sebelumnya, di sini kita menggunakan `CachedNetworkImage`.
