# Mengubah Product Provider

---

Pertama, mari kita ubah product provider. Di sini, kita akan menambahkan sebuah fungsi yang dapat melakukan toggle untuk mengubah status `isFavorit`. Silakan sesuaikan source code-nya menjadi seperti ini:

```dart
    ),
    ProductModel(
      id: "id-4",

      price: 250000,
      imageUrl:
          "https://images.pexels.com/photos/585752/pexels-photo-585752.jpeg?auto=compress&cs=tinysrgb&w=1260&h=750&dpr=1",
    ),
  ];

  List<ProductModel> getProducts() {
    return _products;
  }

  void addProduct(ProductModel product) {
    _products.add(product);
    notifyListeners();
  }

  ProductModel findById(String id) {
    return _products.firstWhere((product) => product.id == id);
  }

  void toggleFavorite(String id) {
    ProductModel selectedProduct = findById(id);

    selectedProduct.isFavorite = !selectedProduct.isFavorite;

    notifyListeners();
  }
}
```

Bisa dilihat, perubahan terjadi dengan cara melakukan inverse dari nilai boolean tersebut.

# Mengubah Product Widget

---

Sekarang, buka `product_widget.dart`, lalu sesuaikan source code-nya menjadi seperti ini:

```dart
import 'package:flutter/material.dart';
import 'package:provider/provider.dart';

import '../models/product_model.dart';
import '../providers/product_provider.dart';
import '../screens/product_detail_screen.dart';

class ProductWidget extends StatelessWidget {
  final ProductModel product;

  const ProductWidget({
    Key? key,
    required this.product,
  }) : super(key: key);

  @override
  Widget build(BuildContext context) {
    return ClipRRect(
      borderRadius: BorderRadius.circular(10.0),
      child: GridTile(
        footer: GridTileBar(
          backgroundColor: Colors.black54,
          leading: Consumer<ProductProvider>(builder: (context, value, child) {
            return IconButton(
              onPressed: () {
                context.read<ProductProvider>().toggleFavorite(product.id);
              },
              icon: Icon(
                product.isFavorite ? Icons.favorite : Icons.favorite_border,
                color: Colors.blue.shade300,
              ),
            );
          }),
          title: Text(product.name),
          trailing: IconButton(
            onPressed: () {},
            icon: Icon(
              Icons.shopping_cart,
              color: Colors.blue.shade300,
            ),
          ),
        ),
        child: GestureDetector(
          child: Image.network(product.imageUrl),
          onTap: () {
            Navigator.pushNamed(
              context,
              ProductDetailScreen.routeName,
              arguments: product.id,
            );
          },
        ),
      ),
    );
  }
}
```

Perhatikan pada `IconButton` favorite. Di sana, kita membuat kondisi menggunakan ternary. Artinya, kita menampilkan `favorite_border` jika nilai `isFavorite` adalah false dan menampilkan `favorite` jika nilai `isFavorite` adalah true.

Perhatikan juga, di sana terdapat `Consumer`. Tujuannya adalah agar ketika perubahan terjadi, maka screen akan melakukan render ulang, sehingga perubahan tersebut bisa dilihat.
